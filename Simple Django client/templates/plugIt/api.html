{% trans "API" %}

{% blocktrans %} The API allow a PlugIt server to access different informations, about user or current organization. {% endblocktrans %}

/user/<id>

PlugItAPI.get_user(pk)
Return information about an user.

/orgas/

PlugItAPI.get_orgas(pk)
Return the list of pk/name for all organizations.

/orga/<id>

PlugItAPI.get_orga(pk)
Return information about an organization.

/members/

PlugItAPI.get_project_members()
Return the list of members in the project.

/mail/

PlugItAPI.send_mail(sender, subject, recipients, message, response_id=None, html_message=False)
Send a mail with given parameters. If response_id is set, the sender email will be set by the system and recipents replies will be send back to the PlugIt server. {% endblock %}
